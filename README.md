# Expense Tracker

A full-stack web application for tracking personal expenses with user authentication, 
CRUD operations, category-wise analysis, and visual analytics.

## 🚀 Features

- **User Authentication**: Secure sign-up and login with JWT tokens
- **Expense Management**: Add, view, edit, and delete expenses
- **Category Organization**: Organize expenses by 8+ categories
- **Analytics Dashboard**: View expense distribution with pie charts
- **Responsive Design**: Works on desktop and mobile devices
- **Secure API**: Role-based access control with Spring Security
- **Database**: H2 for development, ready for MySQL in production

## 💻 Technologies

| Layer | Technology | Version |
|-------|-----------|---------|
| **Backend** | Spring Boot | 3.1.0 |
| **Frontend** | HTML/CSS/JavaScript | ES6+ |
| **Database** | H2 (Dev) / MySQL (Prod) | Latest |
| **Security** | Spring Security + JWT | 0.11.5 |
| **ORM** | JPA / Hibernate | Default |
| **Build Tool** | Maven | 3.6+ |

## 📋 Prerequisites

- **Java 17** or higher
- **Maven 3.6** or higher
